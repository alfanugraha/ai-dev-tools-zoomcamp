# AI Dev Tools Zoomcamp

AI Dev Tools Zoomcamp is a free online bootcamp via zoom  that helps you use AI tools to write better code, faster. All materials are obtained from [DataTalks.Club AI Dev Tools Zoomcamp](https://github.com/DataTalksClub/ai-dev-tools-zoomcamp) repository.

## Introduction

This repository contains the materials for the AI Dev Tools Zoomcamp, a free, self-paced online course designed to teach the early career or advanced developers how to build and deploy AI applications using modern tools and frameworks.
